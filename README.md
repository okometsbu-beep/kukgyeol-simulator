# 진심의 증거 · 국제연애 미스터리

국제연애 상대의 말이 진심인지, 현실적인 이해관계인지, 업체가 만든 압박인지, 계획된 사기인지 대화와 증거로 판단하는 모바일 미스터리 게임입니다.

## 이번 개편의 핵심

- 48개 장으로 이어지는 `첫 연락 → 현지 연애 → 사실 확인 → 거주 합의 → 결혼 준비 → 신혼 → 아이 계획 → 출산·육아 → 결혼 1년 뒤 최종 판단` 캠페인
- 수치로 범인을 알려주지 않는 관계 노트: `확인된 사실 / 해석이 필요한 단서 / 미확인 이야기`
- 상대의 설명과 직접 연결되는 자료를 골라 앞뒤가 다른 말을 확인하는 대화 장면
- 틀린 증거 제시, 근거 없는 공개 비난, 잘못된 범인 지목에 큰 관계·엔딩 페널티
- `결혼을 미루는 선택`과 `상대 또는 업체를 사기라고 공개 지목하는 선택`을 분리하고, 결혼을 골라도 게임이 끝나지 않음
- 진심, 현실적 결혼, 번역 오해, 업체 중간 가로채기, 숨은 배우자, 입국·취업 계획, 상대와 업체가 함께 꾸민 사기, 투자형 로맨스 사기, 중복 약혼, 압박에서의 탈출 등 10가지 숨은 사건
- 같은 인물도 판마다 다른 진실을 가질 수 있으며, 국적·외모·문신·야간근무·임신 의향은 범죄 증거로 처리하지 않음
- 국가 6곳 × 얼굴 8명, 총 48개 기본 얼굴. 프로필과 실제 만남은 한 인물로 짝지으며 결혼 뒤에도 같은 실제 얼굴을 유지해 인물 바뀜 현상을 막음
- 이름 96개와 직업·도시·관심사·동기·말투·경계선·언어 수준을 조합해 매 판 새로운 프로필 후보를 생성
- 실제 만남 앱처럼 보이는 프로필 탐색, 멤버십 상품, 가상 카드 결제 확인과 게임 속 잔액 차감
- 나이에 맞춘 새 얼굴을 가진 플레이어 6명과 서로 다른 현금·월수입·월 고정비·직업 이동성·외모·말재주·공감·판단·배짱·약점
- 영상통화와 실제 만남에서 주인공의 외모와 나이 차가 첫인상에 별도로 반영되며, 일본 노선의 큰 나이 차는 높은 외모·신뢰·진심으로 일부만 만회 가능
- 주인공 전체 프로필 창에서 소개·재정·소통 방식·현재 능력·성장 기록 확인
- 능력이 필요한 선택은 성공 확률을 미리 표시하고, 선택하면 1~100 주사위 판정 애니메이션 뒤 성공·실패를 반영. 판정 경험과 자기계발로 능력치도 성장
- 자동번역·음성·영상·독립 통역 중 첫 소통 방식을 고르고, 질문과 답 형식의 채팅 및 서로 다른 상대 반응을 진행
- 상대가 처음부터 계획에 가담했을 가능성은 매 판 정확히 50%로 나뉘며, 극도로 높은 신뢰·호감과 성공 판정에서는 자발적 고백과 계획 중단 가능
- 사적인 신체·건강 정보와 속궁합은 상호 동의한 친밀한 관계 뒤에만 공개되며 단독으로 사기 증거가 되지 않음
- 거주국은 결혼 전에 양쪽 직업·수입·이동성을 놓고 먼저 합의하고, 결혼식 장소 같은 공동 결정도 상대의 동의 판정을 거침
- 각 장면 사이 한 번씩 추가 근무·휴식·언어 공부·외모 관리·데이트를 고르는 자유 행동과 결혼 판단 기한을 표시
- 위험한 선택은 현재 신뢰·갈등·스트레스·빚을 보여 주는 확인창 뒤에만 강행 가능
- 결혼식 장소(현지·한국·양국)와 규모, 생활비 통장, 배우자의 일, 가족 송금·부모 돌봄·양가 방문, 잠자리 갈등, 임신·출산·육아 방식 선택
- 재정 창의 `전체 / 수입 / 지출` 장부에서 모든 현금 흐름을 확인. 결혼 전에는 주인공 생활비·고정비, 결혼 뒤에는 두 사람의 수입과 생활비·가족 지원·아이 양육비를 실제 잔액에 반영
- 여성 파트너의 나이를 임신 가능성의 가장 큰 가중치로 두고 35세 이후 하락 폭을 크게 반영
- 빚 1,200만원, 스트레스 100, 신뢰 0, 갈등 100이면 결혼 전후 어느 때든 게임오버
- 선택지마다 비용, 예상되는 장점과 위험, 능력 판정 확률을 익숙한 표현으로 표시
- 새 매치, 첫 채팅, 첫 만남, 연애 성립, 친밀한 밤, 업체 의심, 결혼, 입국, 출산, 야간업소 관련 컷신
- 기기 내 자동저장과 PWA 홈 화면 설치 지원

## 비용 모델

업체 노선은 공개 조사에서 확인한 평균 업체 비용 약 1,463만원과 부대비용 약 469만원을 기준점으로 삼았습니다. 한국 결혼식은 2025년 공개된 전국 결혼서비스 평균 약 2,091만원을 기준으로, 장소와 규모에 따라 나눴습니다. 영유아 1명당 월 기본 양육비는 2024년 조사 명목비용 78만5천원을 사용합니다. 항공·체류, 통역, 서류 발급·번역, 현지 조사비는 선택에 따라 별도로 빠집니다. 실제 견적과 개인 상황의 차이가 크므로 게임 안 수치는 모두 `게임상 추정치`입니다.

## 실행

별도 빌드가 없는 정적 웹앱입니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173`을 여세요. HTTPS로 배포하면 iPhone의 Safari 공유 메뉴에서 `홈 화면에 추가`해 앱처럼 실행할 수 있습니다.

## 파일 구조

- `index.html`: 화면 구조와 접근성 마크업
- `styles.css`: 모바일 비주얼노벨 UI
- `game-data.js`: 48명 인물과 10개 숨은 사건 데이터
- `game.js`: 선택, 돈, 시간, 관계 노트, 확인 대화, 자유 행동, 저장, 엔딩 엔진
- `assets/`: 가상 성인 인물과 컷신 이미지

## 주의

모든 인물과 사건은 허구입니다. 실제 사례의 구조를 참고했지만 특정 개인이나 채널의 주장을 사실로 재현하지 않습니다. 이 게임은 실제 사람의 진심이나 범죄 가능성을 예측하는 도구가 아닙니다.
