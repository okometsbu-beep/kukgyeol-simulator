# 진심의 증거 · 국제연애 미스터리

국제연애 상대의 말이 진심인지, 현실적인 이해관계인지, 업체가 만든 압박인지, 계획된 사기인지 대화와 증거로 판단하는 모바일 미스터리 게임입니다.

## 이번 개편의 핵심

- 20개 장으로 이어지는 `대화 → 실제 데이트 문답 → 상호 검증 → 조사 → 진술 심문 → 최종 판단` 루프
- 수치로 위험도를 알려주지 않는 사건 수첩: `확인된 사실 / 해석이 필요한 단서 / 미확인 소문`
- 진술과 직접 연결되는 증거를 골라 제시하는 심문 장면
- 틀린 증거 제시, 근거 없는 공개 비난, 잘못된 범인 지목에 큰 관계·엔딩 페널티
- `결혼 거절·보류`와 `상대 또는 업체를 사기라고 공개 지목`을 분리한 최종 선택
- 진심, 현실적 결혼, 번역 오해, 업체 중간 가로채기, 숨은 배우자, 입국·취업 계획, 업체 공모, 투자형 로맨스 사기, 중복 약혼, 압박에서의 탈출 등 10가지 숨은 사건
- 같은 인물도 판마다 다른 진실을 가질 수 있으며, 국적·외모·문신·야간근무·임신 의향은 범죄 증거로 처리하지 않음
- 국가 6곳 × 인물 8명, 총 48개 인물 설정. 한 후보 화면에는 서로 다른 얼굴 4명만 등장
- 플레이어 6명과 서로 다른 현금·월수입·침착함·대화력·약점
- 현금이 부족하면 급전으로 전환, 월 경과 시 게임상 이자와 스트레스 발생, 고금리 급전 1,200만원·침착함 0·신뢰 0이면 도중 게임오버
- 선택지마다 `지금 드는 비용`과 `앞으로 열리거나 위험해지는 것`을 바로 표시
- 연애 성립, 현지 도착, 업체 의심, 결혼, 입국, 출산, 야간업소 관련 컷신
- 기기 내 자동저장과 PWA 홈 화면 설치 지원

## 비용 모델

업체 노선은 공개 조사에서 확인한 평균 계약·성혼 비용 약 1,463만원과 부대비용 약 469만원을 기준점으로 삼았습니다. 항공·체류, 독립 통역, 서류 발급·번역, 현지 조사비는 판의 선택에 따라 별도로 빠집니다. 개별 업체와 시기마다 견적 차이가 크기 때문에 게임 안에서는 모두 `게임상 추정치`로 취급합니다.

## 실행

별도 빌드가 없는 정적 웹앱입니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173`을 여세요. HTTPS로 배포하면 iPhone의 Safari 공유 메뉴에서 `홈 화면에 추가`해 앱처럼 실행할 수 있습니다.

## 파일 구조

- `index.html`: 화면 구조와 접근성 마크업
- `styles.css`: 모바일 비주얼노벨 UI
- `game-data.js`: 48명 인물과 10개 숨은 사건 데이터
- `game.js`: 선택, 돈, 시간, 증거, 심문, 저장, 엔딩 엔진
- `assets/`: 가상 성인 인물과 컷신 이미지

## 주의

모든 인물과 사건은 허구입니다. 실제 사례의 구조를 참고했지만 특정 개인이나 채널의 주장을 사실로 재현하지 않습니다. 이 게임은 실제 사람의 진심이나 범죄 가능성을 예측하는 도구가 아닙니다.
