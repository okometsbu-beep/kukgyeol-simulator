# 국결 생존기

국제연애의 시작부터 결혼 준비와 결혼 후 10년까지 플레이하는 모바일 우선 시뮬레이션입니다.

## 현재 플레이 범위

- 플레이어 6명, 국가 6곳, 만남 경로 5종
- 선택 국가의 후보 4명만 표시
- 만남 8회, 결혼 준비 5회, 결혼생활 20회
- 큰 인물 초상과 대화창을 중심으로 한 모바일 연애 시뮬레이션 화면
- 매 차례 장기 계획 3종과 현재 상황에 맞춰 압축된 선택 3~4개
- 연애 성립 예상 범위·정보 확인 정도·순자금 전망
- 행동마다 지금 얻는 것·잃는 것·장기 결과·반복 페널티 표시
- 실제 견적처럼 판마다 달라지는 비용과 대출·할부·다른 선택 결제 분기
- 앱 연락두절·외부 메신저·업체 추가견적·서류 반려 등 경로별 사건
- 상대방의 독립적인 선택과 숨은 성향
- 모든 후보에게 독립적으로 부여되는 사기 가능성·개인 사정·아이 의향·임신 관련 개인차
- 부부싸움과 배우자 제안에 대한 문답식 대응
- 배우자 수입·개인 지출·송금·가계비가 반영되는 장부
- 결혼 성사와 별개로 부부 관계·경제·자율·안전·가족계획을 평가하는 결말
- AI로 만든 가상 성인 초상·연애·결혼·입국·출산·위기 컷신 자산

## 실행

정적 웹앱이라 별도 빌드가 필요 없습니다. 로컬 서버에서 `index.html`을 열면 됩니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173`을 여세요. HTTPS로 배포하면 홈 화면에 설치할 수 있는 PWA로 동작합니다.

## 주의

게임 속 비용과 확률에는 공개 자료를 바탕으로 한 값과 게임 균형을 위한 가정이 섞여 있습니다. 특정 국적·외모를 성격이나 위험도와 연결하지 않으며, 개인의 실제 결혼 성공 가능성을 예측하는 도구가 아닙니다.
