# 진심의 증거 · 국제연애 미스터리

국제연애 상대의 말이 진심인지, 현실적인 이해관계인지, 업체가 만든 압박인지, 계획된 사기인지 대화와 증거로 판단하는 모바일 미스터리 게임입니다.

## 이번 개편의 핵심

- 남녀 모두 두 가지 `어두운 욕구`를 가짐. 욕구를 말하고 멈춤 규칙을 지키면 그림자 공명과 무한 점수가 쌓이고, 통제·감시·강요로 왜곡하면 별도 위험도가 올라 전용 배드 엔딩까지 이어짐
- 여성마다 밝은 관계 니즈 2개, 금기 1개, 숨은 모순 1개를 부여. 반복 행동으로 그녀의 니즈를 충족하면 상대 궁합 점수가 쌓이며, 말투만 바뀌는 성격이 아니라 선택 결과와 실패 반응도 달라짐
- 상대 프로필을 `기본 정보 / 성격·생활 / 사생활·그림자` 세 탭으로 압축하고, 어두운 욕구 두 개가 모두 열려야 숨은 모순을 공개
- 생성 시 정해지는 외모·나이·신체 조건 점수의 비중을 낮추고, 반복 행동으로 충족하는 상대 니즈와 안전한 그림자 공명의 비중을 높임
- 장면 풀에서 만남 경로·성격에 맞는 사건을 고르는 가변 캠페인. 교제 1~108일과 결혼 1~24개월의 고정 시점으로 이동해 장면별 일수 중복 누적을 막음
- 상대가 먼저 사소한 일상 메시지를 보내고, 자기 방식의 데이트를 제안하고, 결혼 조건을 꺼내는 세 번의 파트너 주도 장면
- 상대는 처음부터 완성된 프로필이 아님. 만남 경로에서 처음 들은 정보, 반복 행동으로 알게 된 성격, 공식 확인된 사실, 아직 잠긴 사정을 한 인물 정보 창에서 구분
- 연락 리듬·애정 표현·갈등 방식·돈 습관·독립성·질투 지점이 서로 다른 8가지 행동 성격. 각 성격에 첫인사·일상·초대·기쁨·상처·돈·의심·사과·결혼·신혼 전용 말투와 대사를 부여
- 키·체형은 실제 만남, 문신·서로 느낀 친밀감은 상호 동의한 시간, 몸무게·건강은 함께 받은 상담, 구체적인 신체 정보는 결혼 뒤 높은 신뢰와 자발적 공유가 있어야 열리는 단계별 잠금
- 개인 빚·과거 관계·가족 채무·아이·직업 차이·체류 관심 같은 숨은 사정은 단순 호감 반복으로 열리지 않으며 `대화 / 충분한 신뢰 / 공식 자료 / 관련 사건` 중 정해진 조건이 필요
- 별도 노트 화면 없이 상대 프로필 안에 `처음 들은 소개 / 행동으로 드러난 성격 / 동의해 알게 된 사생활 / 발견 기록과 확인 자료`를 통합
- 잠긴 상대 정보는 항목 이름까지 드러내지 않는 암흑 슬롯으로 표시하고, 새 정보가 열릴 때 프로필 해금 효과와 사실 팝업을 재생
- 남성 주인공마다 외모·체형·성격·나이 차·자녀 수·생활 방식이 다른 니즈 5개를 부여. 관련 정보가 실제로 열린 뒤 조건이 맞으면 특별점수를 지급하며 총점에는 만점이 없음
- `그녀가 나를 믿는 정도`와 `내가 사실을 확인한 정도`를 별도 수치로 분리. 다정한 선택만 반복해도 미스터리의 빈칸은 자동으로 열리지 않음
- 조사원을 고용하거나 디지털 확인을 시작하면 직장·계좌·업체, 사진·메시지·사이트 중 조사 대상을 직접 정하는 조사 계획 단계
- 상대의 설명과 직접 연결되는 자료를 골라 앞뒤가 다른 말을 확인하는 대화 장면
- 틀린 증거 제시, 근거 없는 공개 비난, 잘못된 범인 지목에 큰 관계·엔딩 페널티
- `결혼을 미루는 선택`과 `상대 또는 업체를 사기라고 공개 지목하는 선택`을 분리하고, 결혼을 골라도 게임이 끝나지 않음
- 진심, 현실적 결혼, 번역 오해, 업체 중간 가로채기, 숨은 배우자, 입국·취업 계획, 상대와 업체가 함께 꾸민 사기, 투자형 로맨스 사기, 중복 약혼, 압박에서의 탈출 등 10가지 숨은 사건
- 같은 인물도 판마다 다른 진실을 가질 수 있으며, 국적·외모·문신·야간근무·임신 의향은 범죄 증거로 처리하지 않음
- 국가 6곳 × 얼굴 8명, 총 48개 기본 얼굴. 프로필과 실제 만남은 한 인물로 짝지으며 결혼 뒤에도 같은 실제 얼굴을 유지해 인물 바뀜 현상을 막음
- 이름 96개와 직업·도시·관심사·동기·말투·경계선·언어 수준을 조합해 매 판 새로운 프로필 후보를 생성
- 앱은 데이팅 프로필·멤버십, 지인 소개는 소개 카드·식사 비용, 업체는 상담 파일·계약금, 교류 모임은 참가자 목록·참가비처럼 경로마다 다른 후보·결제 UI
- 나이에 맞춘 새 얼굴을 가진 플레이어 6명과 서로 다른 현금·월수입·월 고정비·직업 이동성·외모·말재주·공감·판단·배짱·약점
- 영상통화와 실제 만남에서 주인공의 외모와 나이 차가 첫인상에 별도로 반영되며, 일본 노선의 큰 나이 차는 높은 외모·신뢰·진심으로 일부만 만회 가능
- 주인공 전체 프로필 창에서 소개·니즈·무한 누적 점수·재정·소통 방식·현재 능력·성장 기록 확인
- 대부분의 대화 선택에 능력과 소통 점수를 합친 성공 확률을 미리 표시. 남성 캐릭터별 전용 선택은 필요한 위기 장면에만 나타나며 캠페인당 최대 3회, 보상과 기본 확률도 일반 선택과 경쟁하도록 조정
- 언어가 잘 통하지 않으면 소통 점수가 음수로 시작해 성공 확률이 하락. 전문 통역은 계약·법률·돈 대화에서 강하지만 둘만의 친밀한 대화에서는 확률 상승이 작고 자연스러움·친밀감 손해가 생기며, 사용 자체로 점수를 주지 않음
- 자동번역·음성·영상·독립 통역 중 첫 소통 방식을 고르고, 질문과 답 형식의 채팅 및 서로 다른 상대 반응을 진행
- 상대가 처음부터 계획에 가담했을 가능성은 매 판 정확히 50%로 나뉘며, 극도로 높은 신뢰·호감과 성공 판정에서는 자발적 고백과 계획 중단 가능
- 사적인 신체·건강 정보와 속궁합은 상호 동의한 친밀한 관계 뒤에만 공개되며 단독으로 사기 증거가 되지 않음
- 거주국은 결혼 전에 양쪽 직업·수입·이동성을 놓고 먼저 합의하고, 한국 정착·현지 1년·장거리 생활에 따라 결혼 뒤 주거 선택과 비용표가 실제로 달라짐
- 상단에는 의미가 모호한 `5/48` 진행률 대신 `교제 n일째 / 결혼 n개월째`를 표시하고, 장면 사이 별도 행동 메뉴는 제거
- 확인 대화에서 추가 질문은 진술마다 한 번만 가능하며 이후에는 관련 자료를 제시하거나 이번 설명을 믿어야 함
- 위험한 선택은 현재 신뢰·갈등·스트레스·빚을 보여 주는 확인창 뒤에만 강행 가능
- 결혼식 장소(현지·한국·양국)와 규모 외에 혼인신고 우선·잔치 연기 경로를 추가해 주거 비상금을 지키는 선택 제공
- 한국 정착은 서울 월세·매매와 LTV·DSR·대출 상한을, 현지 거주는 국가별 게임상 생활비 지수와 1년 임대를, 장거리는 두 집·항공비를 계산. 자기자본이 부족한 집은 거짓 성공률 대신 선택 불가로 표시
- 주거 합의 실패 시 선택했던 같은 집으로 강제 입주하지 않고 별도의 한 달 임시 숙소로 이동하며, 집의 크기와 월 부담이 배우자 호감·갈등·피로에 매달 반영
- 30·90·180일 선택의 시간이 더 이상 3일로 잘리지 않으며 실제 개월만큼 수입·생활비·이자·피로가 함께 흐름
- 재정 창의 `전체 / 수입 / 생활 지출 / 주거 / 빚 상환` 장부와 6개월 공동생활 전망에서 현금·보증금·집값·주담대·급전을 분리 확인. 배우자 취업 공백, 임신·육아, 양가 왕래, 실직·휴직·자영업 적자·프리랜서 수입 급락을 월별 반영
- 여성 파트너의 나이를 임신 가능성의 가장 큰 가중치로 두고 35세 이후 하락 폭을 크게 반영
- 빚 1,200만원, 스트레스 100, 신뢰 0, 갈등 100이면 결혼 전후 어느 때든 게임오버
- 선택지마다 비용, 예상되는 장점과 위험, 능력 판정 확률을 익숙한 표현으로 표시
- 일반 대화 장면은 선택한 인물과 짝지어진 생활 사진을 사용하고, 새 매치·첫 만남·연애 성립·결혼·입국·출산 같은 큰 사건은 선택 인물의 얼굴을 컷신 위에 유지
- 기기 내 자동저장과 PWA 홈 화면 설치 지원

## 비용 모델

업체 노선은 공개 조사에서 확인한 평균 업체 비용 약 1,463만원과 부대비용 약 469만원을 기준점으로 삼았습니다. 한국 결혼식은 2025년 공개된 전국 결혼서비스 평균 약 2,091만원을 기준으로, 장소와 규모에 따라 나눴습니다. 영유아 1명당 월 기본 양육비는 2024년 조사 명목비용 78만5천원을 사용합니다. 2026년 7월 주거 선택은 서울 시장 수준을 단순화한 게임상 추정 시세와 수도권 주택구입 목적 주담대 6억원 상한, LTV 70%, DSR 40%, 디딤돌 상품 한도를 반영합니다. 항공·체류, 통역, 서류 발급·번역, 현지 조사비는 선택에 따라 별도로 빠집니다. 실제 시세·대출 가능액·개인 상황의 차이가 크므로 게임 안 수치는 모두 `게임상 추정치`입니다.

## 실행

별도 빌드가 없는 정적 웹앱입니다.

```bash
python3 -m http.server 4173
```

브라우저에서 `http://localhost:4173`을 여세요. HTTPS로 배포하면 iPhone의 Safari 공유 메뉴에서 `홈 화면에 추가`해 앱처럼 실행할 수 있습니다.

## 파일 구조

- `index.html`: 화면 구조와 접근성 마크업
- `styles.css`: 모바일 비주얼노벨 UI
- `game-data.js`: 48명 기본 얼굴, 행동 성격 8종, 남녀 어두운 욕구 10종, 10개 숨은 사건 데이터
- `game.js`: 선택 확률, 통역의 장단점, 무한 점수, 그림자 공명·위험, 돈·주거·대출, 실제 시간, 상대 프로필, 확인 대화, 저장, 엔딩 엔진
- `assets/`: 가상 성인 인물과 컷신 이미지

## 주의

모든 인물과 사건은 허구입니다. 실제 사례의 구조를 참고했지만 특정 개인이나 채널의 주장을 사실로 재현하지 않습니다. 이 게임은 실제 사람의 진심이나 범죄 가능성을 예측하는 도구가 아닙니다.
